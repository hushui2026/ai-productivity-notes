还在手动把 AI 的大段废话复制到 Excel 里整理？每次都要花半小时调整格式？

用**结构化输出 Prompt**，直接让 AI 返回你想要的格式，复制粘贴就能用，彻底告别排版苦力。

很多人用 AI，只关注让它“写什么”，却忽略了“怎么呈现”。结果就是，AI 啰里啰嗦写了一堆，你还得手动剥离关键信息。

想要 AI 乖乖交出可以直接用的数据，只需掌握以下三个步骤。

### ① 明确指定输出载体

不要对 AI 说“帮我整理一下”，要直接下达格式指令。

常见的载体包括：Markdown 表格、CSV、JSON、列表等。

你可以说：“请以 Markdown 表格形式输出”，或者“请以 CSV 格式输出，逗号分隔”。

[截图：普通文本输出 vs 表格格式输出对比]

### ② 定义字段与数据类型

光有表格不够，你还得告诉 AI 表头是什么，每个格子填什么类型的数据。

比如你要提取竞品手机的信息，就要明确规定：价格必须是数字，上市时间必须是年月日。

这样能极大减少 AI 的自由发挥空间，保证数据一致性。

### ③ 终极武器：用代码块锁定结构

如果你要把数据接入代码，或者要导入到数据库，**JSON** 是最好的选择。

这时候，直接在 Prompt 里把 JSON 的 Schema（数据结构）给 AI，让它往里填空。

这是最稳妥的结构化输出方式，来看下面这个实战模板。

```text
你是一个信息提取专家。请从以下文本中提取关键信息，并严格按照 JSON 格式输出。

要求：
1. 输出必须是一个合法的 JSON 对象。
2. 不要包含任何解释性文字，只输出 JSON 代码块。
3. 字段说明：
   - "product_name": 产品名称（字符串）
   - "price": 价格（数字，单位为元，去除货币符号）
   - "features": 核心卖点（字符串数组，最多3个）
   - "release_date": 上市时间（格式为 YYYY-MM-DD，若未提及则为 null）

文本：
[在此输入你的文本]
```

[截图：使用上述 Prompt 后 AI 返回的纯净 JSON 代码块]

按照这个模板，AI 会直接输出一段干干净净的 JSON 数据，没有“好的，这是为您提取的信息”这种废话。

### 效果对比

传统方式：AI 输出大段文本 -> 手动复制到表格 -> 逐行分列调整格式，耗时 **15 分钟**。

AI 结构化方式：直接输出表格或 JSON -> 复制粘贴导入，耗时 **10 秒**。效率提升 90 倍，且零格式错误。

### 小贴士

1. 防废话必杀技：在 Prompt 末尾加上“**只输出目标格式，不要包含任何前置或后置的解释说明**”，能解决 90% AI 爱说废话的问题。
2. 避坑指南：如果要求输出 JSON，尽量要求 AI 输出在代码块中（```json ... ```），这样方便你一键复制，且不会被 markdown 渲染错乱。
