# Ollama 上手：在电脑上本地运行 AI 大模型

上周部门开会，老板突然强调纪律，说以后涉及客户数据的内部资料，谁也不准往线上大模型里塞。这可要了命了，我现在写个周报、整理个竞品分析都离不开 AI，突然让我断网干活，那效率简直直线下降。

后来听技术部的同事说，可以用 Ollama 在自己电脑上跑本地大模型，数据不出电脑，绝对隐私。我想着这感情好啊，赶紧去官网下了个包折腾起来。结果第一天就把自己搞破防了，踩了一堆莫名其妙的坑，差点把电脑都搞死机。

今天就跟大家吐吐槽，我折腾 Ollama 本地部署踩过的几个大坑，你们要是也想搞隐私办公，千万别再走我的老路。

第一个坑，就是被所谓的“傻瓜式安装”给骗了。

其实 Ollama 下载安装确实简单，去官网下个安装包，双击下一步就完事了。装完之后，我看网上教程说直接在终端里输入命令就能跑模型，心里还想着这不挺容易嘛。

我当时直接敲了个 `ollama run llama3`。好家伙，回车按下去，电脑风扇直接开始起飞，声音大得像直升机要起飞一样。然后屏幕卡得鼠标都动不了，等了十几分钟，模型加载是加载出来了，问它一句话，它半天蹦出一个字。

我人都傻了，这哪是 AI 提效，这是给电脑上刑啊。

后来去查了才发现，这纯粹是我没搞清楚自己电脑的斤两。`ollama run llama3` 默认拉取的是 8B 参数的版本，对内存和显存都有要求。我这台办公本是 16G 内存，核显，硬跑 8B 模型当然吃不消。

解决办法其实很简单，换个小模型。别去死磕那些大参数的模型，打工人办公用，换个轻量级的就行。比如阿里的 Qwen，或者直接用 Llama 3 的更小参数版本。敲命令的时候直接指定小模型：`ollama run qwen2:1.5b`。换成 1.5b 的模型后，风扇立马不转了，回复速度也肉眼可见地顺手了。大家在跑之前，一定先看看自己电脑几斤几两，8G 内存就老实跑 1.5b 或者 0.5b 的，16G 内存再去碰 7b、8b 的，别盲目贪大。

第二个坑，纯命令行聊天简直是反人类。

模型跑起来后，Ollama 默认的交互界面就是一个黑乎乎的终端窗口。你敢信 2024 年了我还在对着黑框框打字？这倒也罢了，最致命的是，它输出的内容一大段一大段的，在终端里根本没法排版，想复制其中一句话都费劲。

我想让它帮我改一段保密合同条款，结果复制粘贴的时候连带着各种奇怪的符号，还要手动去删，效率没提上去反而降了。讲真，要是一直这么用，我宁愿不用 AI。

后来我发现，Ollama 其实只是个“后端引擎”，它负责在后台默默跑模型，咱们完全可以给它配个好看好用的“前端壳子”。

去搜一个叫 Chatbox 的软件，Win 和 Mac 都有。下载安装好之后，点开软件左下角的设置图标，找到模型提供商，下拉选 Ollama。它一般会自动识别到本地端口，如果没有，就把 API 地址填成 `http://localhost:11434`，再选上你刚才跑起来的那个模型名字。搞定这一步，你就能拥有一个跟网页版 ChatGPT 几乎一模一样的聊天界面了。不仅能正常复制粘贴，还能多开对话窗口，这才叫干活嘛。

第三个坑，拿线上大模型的 Prompt 去套本地模型。

这个坑挺隐蔽的，我一开始都没意识到。工具弄顺手了，我就拿它来写周报。我把之前在网页版用的那套提示词直接扔了进去，让它“扮演资深产品经理，深度分析本周工作，输出结构化周报”。结果它给我吐了一堆假大空的废话，连我们公司的业务专有名词都用错了，逻辑还前后矛盾。

我第一反应是这本地模型太笨了。但仔细一想不对，线上大模型在后台有无数人对它的回答做了微调，你随便给个模糊指令它都能猜个八九不离十。但本地模型是张白纸，它不知道你的上下文，也不知道你要什么风格，你越是用那种宏大的词汇，它越容易抓瞎。

想通了这一点，我换了个思路，把提示词改得极其具体，把背景信息全部喂给它。别说，这一改效果立竿见影，不仅周报写得像模像样，连一些工作上的小细节都帮我梳理得清清楚楚。

给大家分享一个我改出来的通用工作周报 Prompt，亲测好用：

```text
你是我的工作助手，帮我整理本周的工作周报。
以下是本周我做过的零散事项：
[把你的流水账工作内容粘贴在这里]

请按照以下结构输出周报：
1. 本周核心产出：提炼最重要的3个结果
2. 具体进展：按项目分类，用要点说明
3. 遇到的问题及下周计划：列出阻碍和接下来的动作

要求：语气务实，不要客套话，多用动词，避免使用“致力于”、“助力”等虚词。
```

把 Prompt 拆细，把要求说死，本地模型一样能给你干得漂漂亮亮的。它最大的优势就是绝对听话，你给它设好边界，它就不会像某些线上模型那样突然给你来一套道德说教或者废话文学。

折腾完这三个坑，我现在基本上把一部分涉密的内部工作流全挪到了本地模型上。不用担心公司数据泄露，断网也能用，心里踏实多了。
